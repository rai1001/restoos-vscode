# Guia de Visualizacion de Datos - CulinaryOS

## Paleta de colores
- Azul Chef: #378ADD (KPIs, lineas principales)
- Verde Ingredientes: #1D9E75 (Stock ok, ganancia)
- Naranja Alerta: #D85A30 (Avisos moderados)
- Ambar Inventario: #BA7517 (Atencion requerida)
- Rojo Critico: #E24B4A (Alertas, stock agotado)
- Gris Neutro: #888780 (Texto secundario)
